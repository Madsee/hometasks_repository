Первое задание

Правила оформления тегов

В закрывающем теге надо ставить знак /. Значение класса надо писать в кавычках.

Правила оформления селекторов

После названия селектора надо ставить знак {. После значения свойства надо ставить знак ;. В конце селектора надо ставить знак }.

Типичные ошибки

Нет знака / в закрывающем теге. Не установлена кодировка utf-8,из-за этого символы кириллицы не воспринимаются.Значение класса написано без кавычек.

Синтаксис оформления внешнего css файла и как его подключить

Подключить внешний css файл можно через тег link или style

Синтаксис селектора по классу

Синтаксис селектора по классу в html требует определения имени класса и написание его имени между кавычек в атрибуте class. В CSS синтаксисе селектора по классу требует вызова имени класса после точки и указание свойств и их значений в скобках.
