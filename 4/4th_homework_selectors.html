CSS-правила

Весь CSS код состоит из правил. В правиле есть хотя-бы 1 селектор и свойство.

CSS-свойства

В CSS свойства можно разбить на следующие группы:

  • оформление текста;
  • работа с размерами и отступами;
  • позиционирование элементов;
  • создание сеток;
  • декоративные: цвета, фон, тени;
  • другие

Сетка страницы

Веб-сайт состоит из шапки, главного меню, блока с основным содержанием, боковых колонок, подвала. Эти блоки могут быть расположены друг под другом, в несколько колонок или сложнее. Такое расположение блоков сайта называется сеткой.

Каскадность

CSS расшифровывается как каскадные таблицы стилей. Cтили для элемента могут быть определены в нескольких местах. Браузер находит все правила, описывающие элемент, комбинирует их, а затем получает итоговый список свойств для этого элемента. В комбинировании есть четкие правила, которые опираются на приоритетность и специфичность.

Наследование

Наследование заключается в том, что часть стилей передается от родительского элемента к дочерним.

Селекторы

Селекторы указывают к каким элементам применять CSS-свойства. Простейшие селекторы — это селекторы по именам тегов. Такие селекторы содержат имя тега без символов < и >. Сложные селекторы — это селекторы с использованием классов и псевдоклассов. Класс объединяет разные элементы в смысловые группы и применяет к ним одинаковое оформление. Селектор с использованием класса задается так ".класс".

Контекстные селекторы

Такие селекторы состоят из нескольких частей, разделённых пробелом. Их используют для того, чтобы применить стили к элементу, если он вложен в нужный элемент.

Соседние селекторы

Соседние селекторы используются для элементов, расположенных рядом. Соседние селекторы записываются с помощью знака +.

Дочерние селекторы

Дочерние селекторы используются к ближайшему потомку элемента. В дочернем селекторе для записи используется символ ">"

Псевдоклассы

Псевдоклассы — это дополнения к обычным селекторам, которые делают их ещё точнее. Псевдокласс добавляется к селектору c помощью символа ":". Пример - селектор:псевдокласс.

Псевдоклассы 2

Псевдокласс first-child выбирает первый дочерний элемент родителя, псевдокласс last-child выбирает последний дочерний элемент. Псевдокласс nth-child выбирает теги по порядковому номеру, не используя классы. После nth-child в скобках пишется число(номер элемента) или формула. Псевдокласс hover выбирает элемент, если на него наведена мышь. С помощью этого псевдокласса можно создавать динамические эффекты. Псевдоклассы для ссылок link, visited, и active, обозначают посещена ли уже ссылка или эта ссылка активна. Псевдокласс focus выбирает элемент, который находится в фокусе.

Селекторы по id

Этот селектор выбирает элементы с идентификатором. Он записывается так - #идентификатор.
