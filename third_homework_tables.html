Создание таблицы

Для того чтобы создать таблицу, надо внутри тега "table" расположить теги "tr" для создания строк в таблице, а внутри "tr" расположить теги "td" для создания ячеек таблицы, и внутри "td" написать содержимое ячейки.

Рамки

Рамки в таблице можно задать c помощью CSS-свойства border. Создать рамки элемента таблицы только с определенной стороны, можно c помощью CSS-свойств "border-top", "border-bottom", "border-left" или "border-right".

Отступы внутри ячеек

Внутренние отступы можно задать с помощью CSS-свойства "padding" со значением в пикселях или процентах. Создать отступ внутри ячейки только с определённых сторон можно с помощью CSS-свойства "padding-top", "padding-left", "padding-bottom" или "padding-right".

Отступы между ячейками

Отступы между ячейками можно задать c помощью CSS-свойства для таблицы "border-spacing" со значением в пикселях или в процентах.

Семантическая вёрстка

Создание веб-страниц на языке HTML, используя теги в соответствии с их предназначением, и с последовательной иерархией.

Ячейки-заголовки

Ячейки-заголовки создаются с помощью тега "th" внутри тега "tr", и выделяют название строк или столбцов. Текст в таких ячейках отображается жирным шрифтом по центру.

Заголовок таблицы

Заголовок таблицы создается с помощью текста, обёрнутого в тег "caption".Этот тег должен быть внутри "table", и должен быть первым внутри него. Заголовок можно перемещать по вертикали с помощью CSS-свойства "caption-side" со значениями "top" и "bottom", а также можно перемещать по горизонтали с помощью CSS-свойства "text-align" со значениями "left" и "right".

Объединение ячеек в строках

Чтобы объеденить ячейки по горизонтали надо у тегов "td" u "th" использовать атрибут colspan с каким-либо значением.

Объединение ячеек в столбцах

Чтобы объеденить ячейки по вертикали надо у тегов "td" u "th" использовать атрибут rowspan с каким-либо значением.

Оформление таблицы

Для того чтобы задать цвет ячейкам таблицы надо использовать CSS-свойства: background-color-задает цвет фона ячейки; color-задает цвет текста в ячейке; border-color-задает цвет границы ячейки.

Размеры элементов таблицы

Для того чтобы задать ширину или высоту ячейкам или столбцам, надо использовать CSS-свойства "height" и "width".
