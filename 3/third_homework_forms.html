Создание формы

Для того чтобы создать форму, надо поле формы обернуть в тег "form", и у тега "form" задать атрибуты "action" и "method". У "action" задается значение(URL), куда будут отправляться данные. У атрибута "method" задается способ отправки.

Поле формы

Поле формы создается с помощью одиночного тега "input" и у него есть два обязательных атрибута: "type" и "name". Атрибут "type" задает тип формы, как она будет отображаться, по умолчанию тип text, то есть в форму надо бдет вводить текст. Атрибут "name" задает имя формы, и оно должно быть уникальным.

Подпись для поля ввода

Подпись для поля требуется, чтобы связать текст и поле логически. Для того, чтобы это сделать, надо поле обернуть в тег "label".

Связка подписи и поля с помощью "id"

Чтобы связать поле ввода с подписью по "id", надо к полю ввода задать идентификатор c помощью атрибута "id", потом подпись обернуть в "label", к нему добавить атрибут for и атрибуту for задать значение как и в "id". Этот прием используется когда поле ввода и подпись не стоят рядом, например в таблице.

Поле для ввода пароля

В поле для ввода пароля текст будет отображаться звёздочками. Для того чтобы добавить это поле, надо у "input" атрибуту "type" задать значение "password".

Отправка формы

Для того чтобы отправить форму, надо создать кнопку отправки. Она создается с помощью тега "input" со значением "submit". Надпись на конопке можно задать с помощью атрибута "value" у тега "input".

Чекбокс

Поле-переключатель

Переключатель создается с помощью тега "input" с типом "radio". Переключатели используют когда надо выбрать один вариант из списка. У переключателей из одной группы должны быть разные значения, но одно имя и значения задаются с помощью атрибута "value", который для переключателей обязательный.

Раскрывающийся список

Раскрывающийсся список позволяет выбрать один вариант из нескольких. Раскрывающийся список создаётся с помощью парного тега "select" с атрибутами "name" и "id". Варианты ответов задаются с помощью парных тегов "option", распологающихся внутри "select". Внутри "option" задается подпись варианта ответа, а в атрибуте "value" у тега "option" задается значение варианта ответа.

Поле для загрузки файлов

Для того чтобы создать поле для загрузки файлов, надо у атрибута "type" в теге "input", задать значение "file". Чтобы браузер смог передать файл на сервер, надо форме добавить атрибут "enctype" со значением "multipart/form-data"

Скрытое поле

Скрытое поле используется, когда в форме нужно отправить какие-то дополнительные данные, которые не вводятся пользователем. Скрытое поле создается с помощью типа "hidden" тега "input".
